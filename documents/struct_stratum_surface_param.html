Detailed Description

Parameters for generating geological interface surfaces from boundary shapes and a DEM.

Stage 1 derives topological adjacency from the footprint polygons and optionally projects them onto the terrain DEM. Stage 2 extrudes each adjacency wire downward along a configured spine to form a 3-D surface.

◆ boundary

entt::entity StratumSurfaceParam::boundary = entt::null

StructureShape 2-D outer boundary polygon; when set, each interface wire whose endpoints do not reach this polygon is auto-extended along its end-tangent until intersection.

◆ depth

double StratumSurfaceParam::depth = -100.0

Minimum Z-depth (metres, typically negative) that each constructed surface must reach. The spine's lower endpoint is extended downward until its Z <= depth. Default -100 m.

Overshoot multiplier: each boundary/terrain/depth extension is lengthened by (overshootUnits precision) metres beyond the exact intersection point. Typical range 3C5. Default 3.

◆ precision

float StratumSurfaceParam::precision = 1.0f

Spatial precision unit (metres). Acts as the base length for all overshoot calculations. Default 1 m.

◆ skipAdjacency

bool StratumSurfaceParam::skipAdjacency = false

When true, footprint group supplies ready-made boundary Wires; TopoAdjacent (step 1) is skipped. Dip direction is derived from each wire's mid-tangent via right-hand rule.
