Detailed Description

Value-to-color map component used to visualize floating-point scalar properties.

Maps normalized quantity values in [0, 1] to RGBA colors. An optional spatial gradient-transparency map can further modulate opacity based on the local gradient magnitude (primarily for volume rendering).
