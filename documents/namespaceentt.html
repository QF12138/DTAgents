Registers entity lifecycle callbacks and entity constructor for a component.

Template Parameters
CompThe component type to register.
Parameters
[in]ptrPointer to the GraphSystem instance.
[out]dspThe descriptor to populate with callbacks.

This function connects the component's on_construct and on_destroy signals to the GraphSystem's handlers, and sets up an entity constructor callback that creates entities with this component.

Note
The constructor callback creates an entity and emplaces the component.
