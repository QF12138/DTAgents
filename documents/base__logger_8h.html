Detailed Description

Singleton logger wrapping spdlog for application-wide logging.

1.0

This module provides a thread-safe, application-wide singleton logger built on top of the spdlog library. The logger is initialized once on first access and persists throughout the application lifetime.

Features:

  • Singleton pattern ensures a single, consistent logger instance across all modules
  • Thread-safe logging operations via spdlog's internal synchronization
  • Dual output sinks: console (stdout/stderr) and rotating log files
  • Log levels: TRACE, INFO, WARN, ERROR, CRITICAL
  • Supports formatted string messages

Usage:

BaseLogger::info("Application started");
BaseLogger::error("Failed to load config");
BaseLogger::error
static void error(const std::string &msg)
Logs an error-level message.
BaseLogger::info
static void info(const std::string &msg)
Logs an info-level message.

To access the underlying spdlog logger directly for advanced operations:

auto logger = BaseLogger::logger();
logger->set_level(spdlog::level::debug);
BaseLogger::logger
static std::shared_ptr< spdlog::logger > logger()
Returns the underlying spdlog logger for direct use.
