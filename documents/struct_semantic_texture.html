Detailed Description

Stores a raster image and its associated scalar value range.

Used for DEM tiles, geophysical images, and any other 2-D raster data attached to an entity. The image is stored as an OpenCV matrix; the spatial transform is typically carried by a companion GeomPropTrsf2d component on the same entity.
