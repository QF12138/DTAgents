Detailed Description

Multi-stage cascade matrix classification rule.

Stage 0: N input variables -> intermediate class index (N-D lookup). Stage k>0: (previous result, 1 modifier variable) -> new class index (2-D lookup). Out-of-range input values are clamped to the matrix shape.
