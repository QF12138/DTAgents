Detailed Description

A rule that maps inputs to a categorical class.

Three kinds are supported, selected by SemanticRule::kind:

  • Interval (default, legacy): uses from + intervals.
  • WeightedScore: uses weighted.
  • CascadeMatrix: uses cascade.

For Interval, each interval [first, second) maps to the corresponding sub-type index within the to category.
