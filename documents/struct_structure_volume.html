Detailed Description

OpenVDB-based volumetric data component.

Stores up to three named VDB grids:

  • prop ?primary property grid (e.g. rock type, Vp, SDF)
  • mask ?boolean / indicator mask grid
  • norm ?gradient / normal vector grid

The base entity optionally links to an outer boundary volume used as a region-of-interest for interpolation and extraction operations.
