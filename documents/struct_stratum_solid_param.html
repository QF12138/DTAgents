Detailed Description

Parameters for sequential boolean cutting of a solid body with geological surfaces.

When ordered is true the surfaces are assumed to be sorted top-to-bottom; successive Intersection + Difference operations yield N+1 stratum bodies. When false, surfaces are treated as unordered fault planes split via recursive CGAL booleans, yielding up to 2^N independent fragments.
