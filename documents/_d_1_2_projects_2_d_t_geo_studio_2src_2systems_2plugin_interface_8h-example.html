Interface for plugin components that extend the graph system.

Interface for plugin components that extend the graph system. Plugins implement this interface to register new component types with the GraphSystem. Each plugin provides a descriptor containing metadata, callbacks, and serialization functions for its components.

Note
Plugins are discovered and loaded dynamically at runtime.

Q_OBJECT

public:
dt::Descriptor initialize(GraphSystem* ptr) const override {
dt::Descriptor dsp;
dsp.name = "MyComponent";
// ... configure descriptor ...
return dsp;
}
};
