Detailed Description

Parameters for reprojecting a raster from one spatial reference to another.

  • input : entity carrying SemanticTexture + GeomPropTrsf2d
  • srcRef : entity carrying SemanticGeoRef (source CRS); entt::null derived from dataset metadata, or treated as EPSG:4326 when no spatial reference is embedded.
  • dstRef : entity carrying SemanticGeoRef (target CRS); entt::null use the same CRS as the source (no-op warp, but pixelSpacing re-sampling is still applied if set).
  • output : entity produced by the modeller (SemanticTexture + GeomPropTrsf2d)
  • pixelSpacing : target pixel size in target CRS units; 0.0 keep source density.
