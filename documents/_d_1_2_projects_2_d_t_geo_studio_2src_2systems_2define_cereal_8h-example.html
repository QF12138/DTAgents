Registers serialization and deserialization callbacks for a component type.

Registers serialization and deserialization callbacks for a component type.

Template Parameters
CompThe component type to register serialization for.
Parameters
[out]dspThe descriptor to populate with archiver functions.

This function sets up both the archiver (save) and snapshot loader (load) callbacks for the specified component type. The archiver serializes the component to a binary output archive, while the loader deserializes it from a binary input archive.

Note
This function is typically called during system initialization to register all serializable component types.

template <typename Comp>
void loadsave(dt::Descriptor& dsp)
{
dsp.archiver = [](const entt::snapshot& ss, cereal::BinaryOutputArchive& output) -> const entt::snapshot& {
return ss.get<Comp>(output);
};
dsp.ssloader = [](entt::snapshot_loader& ss, cereal::BinaryInputArchive& input) -> entt::snapshot_loader& {
return ss.get<Comp>(input);
};
}
