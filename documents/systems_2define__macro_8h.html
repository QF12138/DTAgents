This file provides cross-platform macros for exporting/importing symbols from the DT Systems DLL. On Windows, it uses __declspec(dllexport) and __declspec(dllimport), while on other platforms it defines an empty macro.

Note
Define NO_DTSYSTEMS_EXPORTS before including this header to disable export decorations entirely.

◆ DTSYSTEMS_EXPORT

#define DTSYSTEMS_EXPORT

Platform-specific export/import macro.

This macro should be placed before class and function declarations to make them exportable from the shared library. The macro automatically selects the appropriate decoration based on the build configuration:

  • DTSYSTEMS_EXPORTS defined: exports symbols (building the DLL)
  • Otherwise: imports symbols (using the DLL)

Usage:

class DTSYSTEMS_EXPORT MyClass { ... };
DTSYSTEMS_EXPORT void myFunction();
