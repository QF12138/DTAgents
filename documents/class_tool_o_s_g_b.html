Clip an OSGB tile tree at a given LOD level by an XY polygon and emit one independent mesh per source texture.

Each distinct texture present in the OSGB tiles at the requested LOD level produces ONE independent StructureMesh — there is NO merging of triangles across textures. The i-th element of meshes is paired with the i-th element of textures (parallel vectors).

Triangles fully outside the boundary are discarded; triangles intersecting it are clipped via OGR Intersection with barycentric UV/normal interpolation.

Parameters
osgOSG model component (PagedLOD descendants are followed).
boundaryXY clipping polygon (Polygon or MultiPolygon).
levelLOD level to extract (root = 0; descend N steps into PagedLOD tree).
meshesOutput: one StructureMesh per distinct texture, in order.
texturesOutput: one SemanticTexture per mesh, same order/length as meshes.
Returns
true on success.

Rasterize an OSGB scene to an ortho-image (DOM) and elevation map (DSM).

Leaf-only PagedLOD descent, mipmap+EWA anisotropic texture filtering, 4x SSAA, and coverage-weighted Z blending. Output extent follows the boundary's envelope; world-space placement is captured in the StructureImage outputs; pixel data is captured in the SemanticTexture outputs (DOM = 3-ch uint8 BGR, DSM = 1-ch float32 with NoData = -9999).

Parameters
osgOSG model component (PagedLOD descendants are followed).
boundaryXY polygon used for both spatial culling and pixel masking.
resolutionWorld-units per pixel.
domImgOutput: DOM placement (location/direction/udirection/size).
domTexOutput: DOM pixels (cv::Mat CV_8UC3, channel = 3).
dsmImgOutput: DSM placement.
dsmTexOutput: DSM pixels (cv::Mat CV_32FC1, channel = 1, noval = -9999).
Returns
true on success.
