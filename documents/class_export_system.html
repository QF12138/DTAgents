Exports geographic and geological (GGE) data to a file.

Parameters
[in]regThe entity registry containing source data.
[in]titleExport title/name.
[in]fpOutput file path.
[in]modalExport modal mode.
[in]srsSpatial reference system for the output.
[in]idsVector of entity IDs to export.
[in]mergeIf true, merges all entities into a single output.
Returns
bool True if export was successful.

Exports spatial entity data (drills, meshes, shapes, etc.) to a geospatial file format using GDAL/OGR. The entities are converted to appropriate geometric representations based on their component types.

Note
The merge parameter affects how multiple entities are combined in the output file.
