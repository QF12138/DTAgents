Detailed Description

Parameters for extracting a vector boundary from a raster image.

Calls ToolImage::Boundary to find the largest contour in the input raster, converts pixel coordinates to world coordinates using the source GeomPropTrsf2d, and outputs an OCCT wire as a StructureShape. When regularize is true, cv::approxPolyDP is applied before coordinate conversion to simplify the polygon with tolerance epsilon (pixels).
