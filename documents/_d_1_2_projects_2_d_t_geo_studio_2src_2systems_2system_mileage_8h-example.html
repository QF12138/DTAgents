System for mileage-based spatial queries and coordinate transformations.

System for mileage-based spatial queries and coordinate transformations. MileageSystem manages mileage-aligned spatial data and provides bidirectional conversion between mileage values (distance along a centerline) and 3D Cartesian coordinates. It supports multiple mileage prefixes, section extraction, and direction queries for engineering applications such as tunnel construction.

Note
Mileage strings follow the format "Prefix+Mileage" (e.g., "DK0+123.45"), where prefix typically ends with 'K' (e.g., "DK0K" is the standard notation).
MileageSystem ms(&registry);
gp_Pnt pos;
if (ms.mileageToPosition("DK0+100", pos)) {
// Use position...
}
