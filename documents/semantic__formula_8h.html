enum class RuleKind : uint8_t
strong

Kind discriminator for SemanticRule.

  • Interval: single continuous input mapped to a category index via half-open intervals (the original behavior, kept in SemanticRule::intervals).
  • WeightedScore: multi-variable weighted scoring (expert-system style), highest score wins, optional minimum threshold and tie-break policy.
  • CascadeMatrix: multi-stage waterfall lookup tables; stage 0 takes N input variables, subsequent stages combine the previous result with one extra modifier variable.
