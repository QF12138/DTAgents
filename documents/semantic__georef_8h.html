Detailed Description

Geospatial coordinate reference system (CRS) component.

Author
WouRaoyu
Date
2026-04-11

A SemanticGeoRef can be specified either via an EPSG code (preferred) or via explicit projection parameters. When epsgCode > 0 the explicit parameters are ignored by the modeller.
