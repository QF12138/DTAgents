Main system class for managing entities, components, and graph visualization.

Main system class for managing entities, components, and graph visualization. GraphSystem is the central orchestrator for the entity-component architecture. It maintains descriptors for all registered component types, manages entity relationships, handles graph layout, and provides signal/slot interfaces for UI updates. The system supports plugins for extending functionality and can serialize/deserialize the entire registry state.

Note
This class inherits from QObject to enable Qt signal/slot functionality for UI integration and event propagation.
