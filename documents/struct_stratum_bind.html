Detailed Description

Configuration for a single stratum layer's extrusion spine.

Stores the sequence order, dip angles at top and bottom, and spine curve type.

Serialisation note: the positional order is (order, dipTop, dipBtm, spineType). Archives written with the old (order, dip) layout load correctly because dipTop maps to the same slot as the former dip field.
