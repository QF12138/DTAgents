Detailed Description

Describes a spatial interpolation job: data source, anisotropy, region, and output.

The interpolation algorithm (IDW, RBF, FDI, Kriging) is stored as a separate parameter component on the same entity. SemanticInterp provides the data plumbing shared by all interpolation methods.
