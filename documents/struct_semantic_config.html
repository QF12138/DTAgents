Detailed Description

Singleton semantic configuration registry stored in the ECS.

Provides named registries for:

  • categorical properties (rock type, weathering grade, ...)
  • continuous scalar properties (Vp, Vs, Young's modulus, ...)
  • derived formulas
  • classification rules

Access the singleton via SemanticConfig::Instance().
