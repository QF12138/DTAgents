◆ ECategory

enum class dt::ECategory : uint8_t
strong

Top-level category for ECS component types.

Used to classify components into logical groups for filtering and dispatch in plugin interfaces and UI panels.

Enumerator
Aspect 

Visual appearance properties (color, transparency, etc.).

Geometry 

Geometric properties (bbox, transform, anisotropy, etc.).

Semantic 

Semantic / metadata components (config, category, formula, etc.).

Structure 

Raw data structures (mesh, points, volume, spine, etc.).

Modeller 

Modelling algorithm parameter components.

Extract 

Extraction algorithm parameter components.

◆ ESubType

enum class dt::ESubType : uint8_t
strong

Sub-category for modeller and extractor ECS component types.

Provides finer-grained classification within ECategory::Modeller and ECategory::Extract for routing tasks to the correct processing system.

Enumerator
None 

Non-logic types (Aspect / Geometry / Semantic / Structure).

ExtGeometry 

Mesh isosurface extraction, point cloud extraction.

ExtSection 

Cross-section / profile extraction.

ExtVolume 

Volume construction / conversion from other data types.

ExtParameter 

Raw survey data parameter extraction (AGP, etc.).

MdlShape 

2-D / 3-D geometric shape construction and boolean ops

MdlMesh 

Mesh modelling and processing (DEM, boolean, remesh).

MdlStratum 

Geological stratum surface and solid modelling.

MdlInterp 

Spatial interpolation and geostatistics (IDW / RBF / Kriging).

MdlRaster 

Raster / image processing and reprojection.

MdlFusion 

Multi-source data fusion and integration.

MdlIntelli 

Machine-learning training, inference, and per-voxel scripts.
