Detailed Description

Filesystem path alias.

System for importing external data into the entity registry.

ImportSystem provides static methods for importing various types of survey and spatial data from files. It supports importing Advanced Geological Prediction (AGP) data and Geographic & Geological & Engineering (GGE) data with support for different exploration methods.

Note
All import methods are static and do not require instantiation.
