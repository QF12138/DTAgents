Detailed Description

Parametric mileage spine (centre-line) for tunnels and linear structures.

Stores an ordered map from mileage distance to 3-D world coordinates, plus optional curve-type annotations and rendering metadata. Profiles and sections reference this spine for sweep and cross-section generation.
