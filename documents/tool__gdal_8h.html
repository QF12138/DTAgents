Returns the RB-cell edge exit index given a quadrant and entry status.

Uses a precomputed table indexed by entry status (corner/vedge/hedge) and quadrant to determine which edge a ray exits through. The "in-face" status (from == 3) must not be passed to this function.

Parameters
quadQuadrant index from PixelVecQuadrant() in [0, 3].
fromEntry status: 0 = corner, 1 = vedge, 2 = hedge.
Returns
Exit edge index.
