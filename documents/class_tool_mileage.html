Detailed Description

Low-level mileage ruler constructed from a StructureSpine centerline.

Wraps a StructureSpine and exposes queries for converting mileage distances to 3-D positions/directions, extracting polylines or typed curve sequences within a mileage range, and reversing from a position back to a mileage value. All geometry is in the coordinate system of the spine's point cloud.

Note
Instances hold a const reference to the underlying StructureSpine; the spine must outlive the ToolMileage object.
See also
MileageSystem for the higher-level multi-spine manager.
