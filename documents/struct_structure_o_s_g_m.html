Detailed Description

Stores a reference to an OSGB scene loaded from disk.

The actual osg::Node lives in AIS_OpenSceneGraph (DTVisualizer). This component only carries the file path and LOD loading parameters; spatial transforms are handled by the companion GeomPropTrsf3d component.
