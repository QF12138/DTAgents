Detailed Description

Parameters for extracting per-parameter data from a raw AGP survey input.

Reads a raw AGP dataset referenced by input, applies mileage information from mileage, optionally constrains to a tunnel geometry via tunnel, and writes one output entity per extracted parameter channel.
