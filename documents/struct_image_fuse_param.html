Detailed Description

Parameters for fusing multiple single-channel raster images into a multi-channel image.

Input images are supplied via a SemanticGroup entity (input). All group members must carry GeomPropTrsf2d + SemanticTexture and share identical geotransform and pixel dimensions. Float (CV_32FC1) inputs are normalized to CV_8UC1 before channel merging.
