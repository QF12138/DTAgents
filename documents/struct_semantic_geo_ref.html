Detailed Description

Full CRS definition component for geo-referenced entities.

Supports three specification modes (in decreasing priority):

  1. epsgCode > 0 look up via the EPSG registry.
  2. wkt non-empty and epsgCode == 0 parse OGC/ISO WKT2 or PROJ string.
  3. Explicit parameters (projType, semi-axes, origin, scale, false origin).
